# 装修进度管理 (pdd-170)

## 项目简介
全栈装修进度管理应用。注册登录后可创建装修项目，设置总预算和开工日期。按阶段（拆除、水电、泥瓦、木工、油漆）管理进度和花费，上传现场照片，记录施工日记，管理材料采购清单，支持邀请家人协作。

## 技术栈
- 前端：React 18 + TypeScript + Vite + TailwindCSS + react-router-dom
- 后端：Node.js + Express + SQLite (better-sqlite3) + TypeScript

## 运行方式

### 后端
```bash
cd backend
npm install
npm run dev
```
后端运行在 http://localhost:3000

### 前端
```bash
cd frontend
npm install
npm run dev
```
前端运行在 http://localhost:5173
